Usage

Click any empty square to place the current player's mark (X or O).

The game automatically checks for a win or a draw after every move.

Click Restart to clear the board and start a new game.

How the game works (brief)

Game board is a 3Ã—3 grid of clickable cells.

A JavaScript currentPlayer variable toggles between 'X' and 'O'.

After each move, the script:

Updates the board UI

Checks all winning combinations

Declares a winner or draw

Disables further moves when game ends
